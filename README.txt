PROGRAMA DE ANÁLISE DE DADOS PESSOAIS:

Este programa em C++ permite realizar uma análise simples de 
dados de um grupo de pessoas. Ele foi desenvolvido para que o 
usuário possa informar informações como nome, sexo, idade e 
altura de várias pessoas e, em seguida, oferece funcionalidades 
para listar os dados, calcular a média das idades, a média das 
alturas e exibir a porcentagem de participantes femininos e 
masculinos.


FUNCIONALIDADES:
- Listar dados: Exibe os dados de todas as pessoas cadastradas 
no sistema;
- Média das idades: Calcula e mostra a média geral das idades 
das pessoas cadastradas;
- Média das alturas: Calcula e mostra a média geral das alturas 
das pessoas cadastradas;
- Porcentagem: Exibe a porcentagem de participantes femininos e 
masculinos em relação ao total;
- Alterar dados: Permite ao usuário alterar informações 
específicas de uma pessoa cadastrada;
- Sair: Encerra o programa.

COMO EXECUTAR O PROGRAMA:
- Clone este repositório em sua máquina local;
- Compile o código-fonte usando um compilador C++ compatível;
- Execute o programa resultante.

REQUISITOS:
- Compilador C++ compatível.

USO:
- Execute o programa;
- Siga as instruções do menu para realizar as ações desejadas.
